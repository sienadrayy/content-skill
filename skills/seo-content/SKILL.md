# SEO Content Skill

Create high-quality, SEO-optimized content that ranks in Google and converts.

## Content Types

### 1. Blog Posts (How-To, Guides, Listicles)
- **Length:** 1500-2500+ words
- **Structure:** Intro → H2 sections → Deep dives → Conclusion + CTA
- **Keywords:** 1 primary + 2-3 secondary
- **Links:** 5-10 internal links, 3-5 external authority links
- **Meta:** Title (60 chars) + Description (160 chars)

### 2. Landing Pages
- **Length:** 800-1200 words
- **Sections:** Headline → Hero → Problem → Solution → Social proof → CTA
- **Keywords:** 1 primary (target keyword)
- **Conversion:** Clear CTA above & below fold
- **Meta:** Exact match title + description

### 3. Meta Descriptions
- **Length:** 155-160 characters
- **Formula:** [Keyword] + value proposition + CTA
- **Example:** "Learn {keyword} in 10 minutes. Step-by-step guide + examples. Start now."

### 4. FAQ Schema Content
- **Format:** 10-15 common questions + answers
- **Keywords:** Long-tail variations + question keywords
- **Rich snippet:** Google may feature in search results
- **Benefit:** Captures "People also ask" section

## SEO Content Framework

### Step 1: Keyword Research
```
Primary: "app store optimization"
Secondary: "ASO guide", "how to optimize app store"
Long-tail: "app store optimization for beginners", "free ASO tools"
```

### Step 2: Content Brief
```
Keyword Focus: "app store optimization"
Search Intent: How-to / Educational
Word Count: 2000
Sections: Definition → Benefits → Strategy → Tools → Checklist
```

### Step 3: Outline (H2/H3 structure)
```
H1: The Complete App Store Optimization Guide
  H2: What is ASO?
    H3: How does it differ from SEO?
  H2: Why ASO Matters
  H2: ASO Strategy
    H3: Keyword Research
    H3: Listing Optimization
    H3: Visual Assets
  H2: Tools
  H2: Checklist
```

### Step 4: Write & Optimize
- Primary keyword in: H1, first 100 words, H2s
- Secondary keywords naturally in body
- Internal links: 5-10 relevant pages
- External links: 3-5 authority sources
- Format: Bold, bullets, shorter sentences

### Step 5: Meta Optimization
```
Title: App Store Optimization Guide - Rank Higher on Google Play (59 chars)
Meta: Learn ASO in 10 minutes. Complete guide + checklist. Start optimizing your app today.
```

## On-Page SEO Checklist

- [ ] Keyword in title (within first 60 chars)
- [ ] Keyword in first 100 words
- [ ] Keyword in H2 headings (2-3 times)
- [ ] Meta description (155-160 chars)
- [ ] Internal links (5-10, keyword-rich anchor text)
- [ ] External links (3-5 authority domains)
- [ ] Images with alt text
- [ ] Mobile responsive
- [ ] Page speed < 2 seconds
- [ ] Readability: short paragraphs, bullets

## Keyword Integration Tips

✅ **DO:**
- Use keyword naturally in sentences
- Vary keyword forms (keyword, keywords, keywording)
- Use secondary keywords & synonyms
- Place keyword in headings

❌ **DON'T:**
- Keyword stuff (unnatural repetition)
- Force keyword into every sentence
- Use exact match everywhere (vary phrasing)
- Ignore user experience for keywords

## Publishing Workflow

1. **Draft:** Write initial content
2. **Internal review:** Check outline, facts, keyword placement
3. **Edit:** Refine prose, fix errors, optimize structure
4. **SEO audit:** Title, meta, headers, links, images
5. **Optimize:** Add internal links, improve readability
6. **Publish:** Create in CMS with metadata
7. **Link building:** Mention other sites, ask for links
8. **Track:** Monitor rankings, clicks, conversion

## Tips for Content Success

1. **Evergreen content:** Topics that stay relevant (guides, how-tos)
2. **Data-backed:** Include stats, studies, original research
3. **Unique angle:** Why should someone read *your* version?
4. **Scannable:** Use headings, bullets, short paragraphs
5. **Call-to-action:** Email list, product page, next article
6. **Update regularly:** Refresh top-performers with new data
7. **Link building:** Reach out to relevant sites for backlinks
