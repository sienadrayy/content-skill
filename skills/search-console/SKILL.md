# Search Console Skill

Optimize organic search performance via Google Search Console.

## Setup

1. **Link Property:** https://search.google.com/search-console
2. **Verify ownership:** Add verification code to site
3. **Wait 48 hours** for data collection

## Key Metrics

| Metric | Meaning |
|--------|---------|
| **Clicks** | Users who clicked your link |
| **Impressions** | Times your link appeared in search |
| **CTR** | Click-through rate (clicks / impressions) |
| **Position** | Average ranking position (1-100) |
| **Coverage** | % of pages indexed vs. blocked |

## Common Analyses

### Top Keywords by Traffic
```
Show keywords ‚Üí impressions ‚Üí clicks ‚Üí position
Identify quick wins (high impressions, low CTR = improve title/meta)
```

### Ranking Improvements
```
Track position changes week-over-week
Find pages moving up in rankings
Understand what's working
```

### Index Coverage Issues
```
Monitor excluded pages
Fix crawl errors
Claim more Google visibility
```

### Mobile Performance
```
Mobile usability issues
Mobile usability metrics
Mobile-first indexing status
```

## Optimization Tips

1. **Improve CTR:** Update meta titles/descriptions for high-impression, low-CTR keywords
2. **Target Long-tail:** Keywords with 0 clicks but impressions are easy wins
3. **Fix Coverage Issues:** Ensure all important pages are indexed
4. **Mobile First:** Optimize for mobile - Google prioritizes mobile indexing
5. **Seasonal:** Plan content around seasonal keyword trends
