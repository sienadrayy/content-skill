## Keyword Research

### Finding Keywords
1. **Tool:** App Annie, Sensor Tower, Mobile Action
2. **Metrics:** Search volume, keyword difficulty, competition
3. **Strategy:** Target 3-5 high-volume keywords + long-tail variations

### Keyword Categories
- **Primary (1):** Main keyword, highest volume
- **Secondary (2-3):** Related keywords, medium volume
- **Long-tail (4+):** Specific phrases, low competition

### Keyword Research Example
```
Primary: "fitness tracker"
Secondary: "calorie counter", "workout app"
Long-tail: "free fitness tracker app", "best workout counter"
```

## Listing Optimization Checklist

- [ ] Title: Primary keyword + brand
- [ ] Short desc: Benefit + secondary keyword
- [ ] Full desc: Features, benefits, CTAs, keywords naturally placed
- [ ] Keywords field: All target keywords (space-separated)
- [ ] Icon: Professional, stands out in search results
- [ ] Screenshots: Show core features, benefit-driven captions
- [ ] Feature graphic: Eye-catching, communicates core value
- [ ] Video: 15-30 sec demo of key features
- [ ] Ratings: Encourage users to rate (target 4.5+ stars)

## Competitor Analysis

### What to Analyze
- Competitor keywords (reverse-engineer from their listing)
- Rating + review sentiment
- Screenshot strategy (number, focus, layout)
- Description length + keyword density
- Update frequency + messaging changes

### Tools
- App Annie / Sensortower (keyword tracking)
- Google Play direct search (related/suggested keywords)
- Review analysis tools (understand sentiment)

## A/B Testing

### Test Elements
1. **Icon:** A/B test 2-3 variations over 1 week
2. **Screenshots:** Test benefit-focused vs. feature-focused
3. **Short desc:** Test CTA variations
4. **Title:** Test keyword variations (if not brand-critical)

### Measurement
- Track conversion rate (installs / clicks)
- Compare week-over-week
- Wait 2-4 weeks for data to stabilize

## Monthly Optimization Cycle

```
Week 1: Analyze keywords + competitor listings
Week 2: Update title, description, keywords with top performers
Week 3: Update screenshots/icon based on CTR data
Week 4: Measure results, prepare next iteration
```

## Key KPIs to Track

- **Keyword Rankings:** Position for top 10 keywords
- **Impressions:** Store page visits
- **CTR:** Clicks / impressions
- **Conversion Rate:** Installs / clicks
- **Retention:** Day 1 / Day 7 retention rate
- **Rating:** Average rating + trend
